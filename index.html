Built for humans and AI agents

Click a button, or call it from code. Same tool, either way.

Every tool here runs as an Apify Actor — which means it's also a remote MCP server. An AI agent can call it directly inside a conversation, with the same typed inputs and outputs a person gets in the browser. No separate "API mode" to configure.

  • One atomic action per call, typed inputs and outputs
  • Works with Claude, Cursor and any MCP-compatible client
  • Pay-per-event pricing — an agent only spends on results it actually gets
